# FrontEnd - Encuestas


Aplicación frontend desarrollada con Vite + React + TypeScript, diseñada para interactuar con el backend de encuestas y mostrar resultados de manera visual mediante gráficos.
Se conecta con el Backend:
https://github.com/Andrezgrondona/surveys-backend



# Tecnologías

Tecnologías utilizadas

Vite – 

React + TypeScript – Librería para interfaces dinámicas y tipado estático

Tailwind CSS – Framework CSS para un diseño responsivo y moderno

Recharts – Librería para visualización de datos (gráficos )

Axios – Cliente HTTP para la comunicación con el backend

#  Rutas Navegacion

-/admin	Panel administrativo para crear, editar y eliminar encuestas

-/surveys	Página principal donde los usuarios pueden responder las encuestas

-/results	Panel de visualización de resultados con gráficos
